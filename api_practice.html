The API For Good Laugh

This API generates a random joke! A very simple and very amusing API. It generates both the setup and the punchline for a joke.

The source of this API is: https://dev.to/alanjc/5-simple-to-use-apis-for-beginners-2e0n

However, the actually API can be found on github at: https://github.com/15Dkatz/official_joke_api

I used the setup and punchline options from the API below.

This API can be used by dads around the world when their joke stash is running low. Another use for this API is to get a quick laugh to cheer you up!
